void anonymous_namespace{TriangulateProcess.cpp}::NGONEncoder::ngonEncodeTriangle ( aiFace *  tri)
inline

Encode the current triangle, and make sure it is recognized as a triangle.

This method will rotate indices in tri if needed in order to avoid tri to be considered part of the previous ngon. This method is to be used whenever you want to emit a real triangle, and make sure it is seen as a triangle.

Parameters
triTriangle to encode.
